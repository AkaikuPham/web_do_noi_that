- Cài đặt xampp có php phiên bản 8.0
- Cài đặt composer
- bật xampp lên start Apache và MySQL
- Bật MySQL và tạo database có tên chung_hiep_shop
- import file chung_hiep_shop_db.sql trong folder db
- Mở terminal run lệnh:
- `php artisan serve`
